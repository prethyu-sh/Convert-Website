## 🧰 Features (Tools)

### Image Tools

* Image compressor (quality control)
* Image to PDF converter
* PDF to Image converter
* Resize image (width/height input)
* Crop image

#### 🔁 Image Format Converter (Bi-directional)

Support conversion **both ways (vice versa)** between formats:

* PNG ↔ JPG/JPEG
* PNG ↔ WEBP
* JPG ↔ WEBP
* WEBP ↔ AVIF
* JPG ↔ AVIF
* PNG ↔ AVIF

👉 Users should be able to:

* Upload any supported format
* Select output format
* Convert instantly

---

### PDF Tools

* Compress PDF
* Merge PDF
* Split PDF

#### 🔁 PDF Conversion (Bi-directional)

Convert **to and from PDF**:

**From PDF:**

* PDF → Word
* PDF → Excel
* PDF → PowerPoint
* PDF → JPG

**To PDF:**

* Word → PDF
* Excel → PDF
* PowerPoint → PDF
* JPG / PNG / WEBP → PDF

## ⚙️ Conversion Logic Requirement

* Automatically detect input file type
* Allow user to select output format
* Use a unified conversion handler instead of separate APIs for each format
* Example API:

  * POST /convert-file

    * input: file + targetFormat
    * output: converted file

👉 Avoid creating too many separate endpoints — use a flexible system

---

## ⚙️ Backend Requirements

* Create REST APIs for each tool:

  * POST /compress-image
  * POST /image-to-pdf
  * POST /resize-image
  * POST /convert-image
  * POST /compress-pdf
  * POST /merge-pdf
  * POST /split-pdf
  * POST /pdf-to-jpg
  * etc.

* Use multer for file uploads

* Store files temporarily

* Delete files after processing (important)
